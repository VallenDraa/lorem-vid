# lorem-vid

installing dependencies:

```bash
bun install
```

running development server:

```bash
bun dev
```

running in production:

```bash
bun start
```

## tech stack

- [bun](https://bun.sh)
- [express.js](https://expressjs.com/)
- [ejs](https://ejs.co/)
- [alpine.js](https://alpinejs.dev/)
- [tailwind.css](https://tailwindcss.com/)
- [fly.io](https://fly.io/)
